# Emacs Dotfiles
This is my personal Emacs Dotfiles. I use 
`straight.el` and `use-package`

![Preview of my dotfiles](images/preview.png)

## Installation
before installing dotfiles, do a bakcup
```bash
git clone https://github.com/Ianiksdf/Emacs-Dotfiles.git ~/.config/emacs
````

## Config structure 
```text
├── early-init.el
├── init.el
├── lisp/
│   ├── my-coding.el
│   ├── my-core.el
│   ├── my-funcs.el
│   ├── my-magit.el
│   ├── my-navigator.el
│   ├── my-productivity.el
│   └── my-ui.el
└── README.md
```
## Costume functions (Scaffolding)
Scaffoloding is just isolated consult dir which behaves like a project manager for default directorys following a hierarchy
